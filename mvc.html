Modelo Vista Controlador

El Modelo Vista Controlador (MVC) es un patrón de diseño de software comúnmente utilizado para desarrollar interfaces de usuario. En MVC, los datos de la aplicación (Modelo), la interfaz de usuario (Vista) y la lógica de control (Controlador) están separados en tres componentes distintos, lo que facilita la modularidad y la reutilización del código.
